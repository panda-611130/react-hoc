### React-Hoc
该工程使用 react 高阶组件知识点，将“商品发布页面” 所需要填写的各种协议基本单元进行拆分，并根据不同业务线协议的差异动态 引入所需要的 form 单元，完成数据的搜集/回填。
各个组件进行独立的数据管理，进行采集数据时候各个组件可以通过自定义 getData 方法对搜集上来的数据进行处理。
